## 1. 개요

### 1.1 기능 소개

Task Detail 패널은 orchay 웹 모니터링 UI에서 선택한 Task의 상세 정보를 표시하는 패널입니다. TSK-06-02 개선을 통해 다음 기능이 추가되었습니다:

- **카드 기반 UI**: 정보를 논리적 섹션으로 그룹화
- **워크플로우 스테퍼**: 현재 진행 단계를 시각적으로 표시
- **진행률 바**: 전체 진행률을 퍼센트로 표시
- **요구사항/기술 스펙 섹션**: PRD 요구사항과 기술 스펙을 패널 내에서 확인
- **접기/펼치기 기능**: 필요한 정보만 펼쳐서 보기

### 1.2 대상 사용자

| 사용자 | 주요 활용 |
|--------|----------|
| 개발자 | Task 상세 정보 확인, 요구사항 및 기술 스펙 참조 |
| PM/관리자 | 프로젝트 진행 상태 모니터링 |

## 2. 시작하기

### 2.1 사전 요구사항

- orchay 웹 서버 실행 중
- 웹 브라우저 (Chrome, Firefox, Edge 권장)

### 2.2 접근 방법

1. orchay 웹 UI 접속 (`http://localhost:8080`)
2. 좌측 WBS 트리 패널에서 Task 클릭
3. 우측 Detail 패널에 상세 정보 표시

## 3. 사용 방법

### 3.1 기본 정보 카드

Task 선택 시 상단에 기본 정보 카드가 표시됩니다.

**표시 내용:**
- **Task ID 배지**: `[프로젝트] [카테고리] [Task ID]` 형태
- **제목**: Task 이름
- **우선순위**: 아이콘으로 표시 (critical, high, medium, low)
- **담당자**: 할당된 담당자
- **태그**: 관련 키워드
- **의존성**: 선행 Task 목록

```
┌─────────────────────────────────────────┐
│ [orchay_web] [development] [TSK-06-02] │
│ Task Detail 패널 개선                   │
│                                         │
│ Priority: ●● high    Assignee: -       │
│ Tags: [detail] [card] [stepper] [ui]   │
│ Depends: TSK-03-01                     │
└─────────────────────────────────────────┘
```

### 3.2 워크플로우 스테퍼

진행 상태 섹션에서 현재 워크플로우 단계를 확인할 수 있습니다.

**단계 구성:**
1. 시작 전 (Todo)
2. 설계 (Basic Design, Detail Design)
3. 승인 (Approved)
4. 구현 (Implement)
5. 검증 (Verify)
6. 완료 (Done)

**시각적 표시:**
- 완료된 단계: 녹색 체크마크
- 현재 단계: 파란색 하이라이트
- 미래 단계: 회색

### 3.3 진행률 바

스테퍼 하단에 진행률 바가 표시됩니다.

**상태별 진행률:**
| 상태 | 진행률 |
|------|--------|
| `[ ]` Todo | 0% |
| `[bd]` 기본설계 | 15% |
| `[dd]` 상세설계 | 25% |
| `[ap]` 승인 | 40% |
| `[im]` 구현 | 60% |
| `[vf]` 검증 | 80% |
| `[xx]` 완료 | 100% |

### 3.4 요구사항 섹션

요구사항 섹션을 펼치면 PRD 정보를 확인할 수 있습니다.

**표시 내용:**
- **PRD 참조**: PRD 문서의 관련 섹션 (예: "PRD 3.6.4")
- **요구사항**: 구현해야 할 기능 목록
- **인수 조건**: 기능 완료 기준

**펼치기/접기:**
- 섹션 헤더 클릭으로 토글
- 기본 상태: 접힘

### 3.5 기술 스펙 섹션

기술 스펙 섹션에서 구현 세부 사항을 확인합니다.

**표시 내용:**
- **tech-spec**: 사용 기술 및 라이브러리
- **api-spec**: API 엔드포인트 정의
- **ui-spec**: UI 스타일 가이드

## 4. FAQ

### Q1: 섹션이 펼쳐지지 않아요

**원인**: JavaScript가 비활성화되었거나 로드 실패

**해결**:
1. 브라우저 JavaScript 활성화 확인
2. 페이지 새로고침 (F5)
3. 브라우저 콘솔에서 오류 확인

### Q2: Task 정보가 업데이트되지 않아요

**원인**: 자동 갱신 간격 (5초)이 지나지 않음

**해결**:
1. 5초 대기 후 자동 갱신
2. 다른 Task 클릭 후 다시 선택
3. 페이지 새로고침

### Q3: 요구사항/기술 스펙이 표시되지 않아요

**원인**: WBS 파일에 해당 정보가 없음

**해결**:
- wbs.md 파일에 `requirements:`, `tech-spec:` 등 섹션 추가 필요

## 5. 문제 해결

### 5.1 일반 문제

| 증상 | 원인 | 해결 방법 |
|------|------|----------|
| Detail 패널 비어있음 | Task 선택 안됨 | WBS 트리에서 Task 클릭 |
| 스테퍼 단계 이상 | 잘못된 status 값 | wbs.md 상태 코드 확인 |
| 접기/펼치기 안됨 | JS 오류 | 콘솔 오류 확인 후 새로고침 |

### 5.2 성능 문제

| 증상 | 원인 | 해결 방법 |
|------|------|----------|
| 패널 로드 느림 | 네트워크 지연 | 서버 연결 상태 확인 |
| 애니메이션 끊김 | 브라우저 리소스 부족 | 다른 탭 닫기 |
