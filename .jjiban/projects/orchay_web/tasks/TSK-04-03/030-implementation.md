## 2. 문서 변경 내용

### 2.1 README.md 수정 내역

| 섹션 | 변경 내용 | 상태 |
|------|----------|------|
| CLI 옵션 | 웹서버 옵션 추가 (`--web`, `--web-only`, `--port`) | ✅ |
| Web UI (신규) | 시작하기, 옵션 테이블, 기능 설명, 사용 시나리오 | ✅ |
| 아키텍처 | `web/` 모듈 구조 추가 | ✅ |
| 의존성 | fastapi, uvicorn, jinja2 추가 | ✅ |

### 2.2 추가된 CLI 옵션 문서

| 옵션 | 설명 | 기본값 |
|------|------|--------|
| `--web` | 스케줄러 + 웹서버 동시 실행 | 비활성화 |
| `--web-only` | 웹서버만 실행 (스케줄링 없음) | - |
| `--port PORT` | 웹서버 포트 번호 | 8080 |

### 2.3 추가된 Web UI 섹션 구조

```
## Web UI
├── 시작하기 (코드 예시 4가지)
├── 웹서버 옵션 (테이블)
├── 웹 UI 기능 (기능 목록)
└── 사용 시나리오
    ├── 시나리오 1: 원격 모니터링
    └── 시나리오 2: 로컬 개발 + 웹 모니터링
```

### 2.4 추가된 아키텍처 항목

```
└── web/             # 웹서버 모듈
    ├── server.py    # FastAPI 앱, 라우트 정의
    ├── static/      # 정적 파일 (CSS, JS)
    └── templates/   # Jinja2 템플릿
        ├── base.html    # 기본 레이아웃
        ├── index.html   # 메인 페이지
        └── partials/    # HTMX 파셜 템플릿
```

### 2.5 추가된 의존성

| 패키지 | 버전 | 용도 |
|--------|------|------|
| fastapi | >=0.115 | 웹서버 프레임워크 |
| uvicorn[standard] | - | ASGI 서버 |
| jinja2 | >=3.0 | HTML 템플릿 엔진 |
