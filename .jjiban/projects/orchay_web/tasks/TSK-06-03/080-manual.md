## 1. 개요

### 1.1 기능 소개

문서 테이블은 Task Detail 패널에서 관련 문서를 테이블 형태로 표시하는 기능입니다. 기존의 단순한 파일명 목록 대신 메타정보(타입, 크기, 수정일)를 함께 제공하여 문서를 더 쉽게 식별할 수 있습니다.

### 1.2 대상 사용자

- **개발자**: Task 관련 설계/구현 문서를 빠르게 참조
- **프로젝트 관리자**: 문서 최신 여부 확인 및 진행 상황 파악

### 1.3 주요 기능

| 기능 | 설명 |
|------|------|
| 문서 테이블 | 문서명, 타입, 크기, 수정일을 테이블로 표시 |
| 접기/펼치기 | 문서 섹션을 토글하여 UI 공간 절약 |
| Document Viewer 연동 | 문서 클릭 시 모달로 내용 표시 |

## 2. 시작하기

### 2.1 사전 요구사항

- orchay 웹서버 실행 (`orchay --web` 또는 `orchay --web-only`)
- 브라우저 접속 (`http://localhost:8080`)

### 2.2 접근 방법

1. 웹 브라우저에서 `http://localhost:8080` 접속
2. 좌측 WBS 트리에서 Task 클릭
3. 우측 Task Detail 패널에서 **"관련 문서"** 섹션 확인

---

## 3. 사용 방법

### 3.1 기본 사용법

#### 문서 목록 확인

1. Task Detail 패널 하단의 **"관련 문서"** 섹션 확인
2. 테이블에서 문서 정보 확인:
   - **문서명**: 파일 이름
   - **타입**: MD(마크다운), IMG(이미지), OTHER(기타)
   - **크기**: 파일 크기 (KB, MB)
   - **수정일**: 마지막 수정 일시

#### 문서 내용 보기

1. 테이블에서 원하는 문서 행 클릭
2. Document Viewer 모달이 열림
3. 마크다운 문서: 렌더링된 내용 표시
4. 이미지 문서: 이미지 표시
5. 닫기: ESC 키 또는 X 버튼 클릭

### 3.2 상세 기능

#### 섹션 접기/펼치기

- **펼치기**: 섹션 헤더의 ▶ 아이콘 클릭 → ▼로 변경, 내용 표시
- **접기**: 섹션 헤더의 ▼ 아이콘 클릭 → ▶로 변경, 내용 숨김
- 기본 상태: 펼침

#### 테이블 정보 이해

| 타입 | 아이콘 | 설명 |
|------|--------|------|
| MD | 📄 | 마크다운 문서 |
| IMG | 🖼️ | 이미지 파일 (png, jpg, gif, webp) |
| OTHER | 📁 | 기타 파일 |

| 크기 형식 | 예시 |
|----------|------|
| 바이트 | 512 B |
| 킬로바이트 | 1.5 KB |
| 메가바이트 | 2.3 MB |

## 4. FAQ

### Q1: 문서가 표시되지 않아요

**A**: Task 문서 폴더(`.orchay/projects/{project}/tasks/{task-id}/`)에 파일이 있는지 확인하세요. 폴더가 없거나 비어있으면 "문서 없음"으로 표시됩니다.

### Q2: 수정일이 잘못 표시되는 것 같아요

**A**: 수정일은 서버의 파일 시스템 시간을 기준으로 합니다. 파일을 복사하거나 이동하면 수정일이 변경될 수 있습니다.

### Q3: 특정 파일이 목록에 없어요

**A**: 숨김 파일(`.`으로 시작)이나 시스템 파일은 목록에서 제외됩니다.

## 5. 문제 해결

### 문제: 문서 클릭 시 모달이 열리지 않음

**원인**: JavaScript 오류 또는 Document Viewer 미로드

**해결 방법**:
1. 브라우저 콘솔(F12)에서 오류 확인
2. 페이지 새로고침 (F5)
3. 캐시 삭제 후 재시도 (Ctrl+Shift+R)

### 문제: 테이블이 비어있음

**원인**: Task 문서 폴더 없음 또는 접근 권한 없음

**해결 방법**:
1. Task 문서 폴더 존재 여부 확인
2. 서버 로그에서 오류 확인
