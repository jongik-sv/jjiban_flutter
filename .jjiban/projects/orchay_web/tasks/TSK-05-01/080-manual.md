# Document Viewer 사용자 매뉴얼

## 1. 개요

### 1.1 기능 소개

Document Viewer는 orchay 웹 모니터에서 Task 관련 문서를 바로 열람할 수 있는 기능입니다.

**주요 기능:**
- 마크다운 문서 렌더링
- Mermaid 다이어그램 시각화
- 이미지 파일 표시 (PNG, JPG, GIF, WebP)
- 모달 팝업 UI

### 1.2 대상 사용자

- orchay 웹 모니터 사용자
- 개발자 (설계 문서, 코드 리뷰 결과 확인)
- PM (프로젝트 진행 상황 파악)

---

## 2. 시작하기

### 2.1 사전 요구사항

- orchay 웹 서버 실행 중
- 브라우저 (Chrome, Edge, Firefox)

### 2.2 접근 방법

1. orchay 웹 모니터 접속 (`http://localhost:8080`)
2. WBS 트리에서 Task 선택
3. Task 상세 패널 하단의 **Documents** 섹션 확인

---

## 3. 사용 방법

### 3.1 문서 열기

1. Task 상세 패널의 **Documents** 섹션에서 문서 클릭
2. 모달 팝업이 열리며 문서 내용 표시

**파일 유형별 아이콘:**
| 아이콘 | 파일 유형 |
|--------|----------|
| 📄 | 마크다운 (.md) |
| 🖼️ | 이미지 (.png, .jpg, .gif, .webp) |

### 3.2 마크다운 렌더링

- 제목, 본문, 코드블록 등 마크다운 문법 지원
- Mermaid 다이어그램 자동 렌더링

**Mermaid 다이어그램 예시:**
```mermaid
flowchart LR
    A[시작] --> B[처리]
    B --> C[완료]
```

### 3.3 이미지 보기

- 이미지 클릭 시 모달에 이미지 표시
- 최대 너비/높이 자동 조절

### 3.4 모달 닫기

모달을 닫는 방법은 3가지입니다:

| 방법 | 설명 |
|------|------|
| ESC 키 | 키보드 ESC 누르기 |
| X 버튼 | 모달 우측 상단 X 클릭 |
| 배경 클릭 | 모달 외부 영역 클릭 |

## 4. FAQ

### Q1. 문서가 표시되지 않습니다.

**A:** 다음을 확인하세요:
- 파일이 `.jjiban/projects/{project}/tasks/{task-id}/` 경로에 존재하는지 확인
- 허용된 확장자인지 확인 (.md, .png, .jpg, .jpeg, .gif, .webp)

### Q2. Mermaid 다이어그램이 렌더링되지 않습니다.

**A:** 마크다운 코드블록에 `mermaid` 언어 태그가 올바르게 지정되어 있는지 확인하세요:

````markdown
```mermaid
flowchart LR
    A --> B
```
````

### Q3. PDF 파일을 열 수 있나요?

**A:** 현재 버전에서는 PDF 파일을 지원하지 않습니다. 향후 업데이트에서 지원 예정입니다.

---

## 5. 문제 해결

### 5.1 "문서를 찾을 수 없습니다" 오류

- 파일이 삭제되었거나 이동되었을 수 있습니다
- Task 디렉토리 경로를 확인하세요

### 5.2 "지원하지 않는 파일 형식입니다" 오류

- 허용된 확장자만 지원합니다: `.md`, `.png`, `.jpg`, `.jpeg`, `.gif`, `.webp`
- PDF, HTML, 실행 파일 등은 보안상 차단됩니다

### 5.3 모달이 닫히지 않습니다

- ESC 키를 눌러보세요
- 브라우저 새로고침 (F5)
